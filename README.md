# Campus Map

A comprehensive map application for Kue campus, built with modern web technologies.

## Overview

Campus Map is a web-based mapping solution designed to help users navigate and explore the Kue campus. The application provides an intuitive interface for discovering campus locations, buildings, and facilities.

## Tech Stack

- **TypeScript** (99.7%) - Primary language for type-safe development
- **JavaScript** (0.3%) - Supporting functionality

## Features

- Interactive campus map visualization
- Location search and discovery
- Building and facility information
- User-friendly navigation interface
- Responsive design

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/hermonbest/Campus-map.git
cd Campus-map
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The application will be available at `http://localhost:3000`
